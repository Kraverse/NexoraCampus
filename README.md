## 📖 Overview

**NexoraCampus** is an AI-powered student community platform that brings everything students need into one modern application.

Instead of switching between multiple apps for internships, notes, events, buying & selling, and peer support, NexoraCampus provides a unified platform where students can discover opportunities, connect with their campus community, and access personalized AI-powered recommendations.

Built with a startup-quality design system, scalable architecture, and intelligent automation, NexoraCampus demonstrates how AI can enhance student engagement and campus collaboration.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/yourusername/NexoraCampus.git
```

Install dependencies

```bash
npm install
```

Configure environment variables

```bash
cp .env.example .env
```

Apply database schema

```bash
npx drizzle-kit push
```

Seed demo data

```bash
npx tsx scripts/seed.ts
```

Start the development server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
